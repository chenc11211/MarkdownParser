# MarkdownParser
MarkdownParser

## 简易Markdown文件解析器

简介：完全使用正则表达式进行查找替换实现的一个简易Markdown文件解析器

功能实现：
* 六级标题（## ）
* 斜体 （*斜体*）
* 加粗 （**加粗**）
* 分割线 （---）
* 链接  （[链接名](地址)）
* 图片 （![title](地址)）
* 行内代码 （`代码`）
* 引用 （> 引用内容）
* 无序列表 （* 列表项）
* 有序列表 （1. 列表项）
* 代码块 （```代码块```）


要点：
* 正则匹配组的使用
* 量词（*[0或多次]，+[1或多次]，？[0或1次]，{n,m}）
* 贪婪量词和惰性量词
* 非捕获性匹配组  (?:\d)
* 正向前瞻与负向前瞻  (?=\d)，(?!\d)
* 非换行符的空白字符[\x20\t\f]
* 所有字符(\s\S),[^]
